A Java-based Weather Tracking Tool that provides real-time weather information for any city. The application fetches weather data such as temperature, humidity, weather conditions, wind speed, and more using a weather API.

## 🚀 Features

- Search weather details by city name
- Display current temperature
- Show humidity levels
- Display wind speed
- Weather condition description
- User-friendly interface
- Real-time weather updates through API integration

## 🛠️ Technologies Used

- Java
- Java Swing (GUI)
- HTTP Requests
- JSON Parsing
- Weather API

## 🎯 How It Works

1. Enter the city name.
2. Click the "Search" button.
3. The application sends a request to the weather API.
4. Weather data is fetched and displayed on the screen.
